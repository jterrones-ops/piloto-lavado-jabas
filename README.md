# Piloto — Lavado de jabas

## Incluye
- Vista Asistente
- Vista Supervisor
- Vista Jefatura
- Apertura de turno
- Confirmación de inicio
- Incidencias con hora inicial y final
- Traslados internos
- Cierre de turno
- Correcciones justificadas
- Historial de cambios
- Productividad automática
- Refrigerio automático de 45 minutos
- Gestión de asistentes
- Reporte CSV

## Ejecutar en PC
1. Instala Python 3.11 o superior.
2. Descomprime la carpeta.
3. Haz doble clic en `iniciar_app.bat`.

## Ver en celular dentro de la misma red Wi-Fi
1. En la PC, abre `iniciar_red_local.bat`.
2. Ejecuta `ipconfig` y busca la Dirección IPv4.
3. En el celular abre: `http://IP-DE-LA-PC:8501`
4. En el menú lateral cambia entre Asistente, Supervisor y Jefatura.

## Nota
Esta es una prueba local. Para acceso real desde cualquier ubicación y para varios usuarios simultáneos, la siguiente etapa debe publicar la aplicación en la nube y usar una base de datos central.
